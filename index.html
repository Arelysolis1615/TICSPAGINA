NOSOTROS

La ingenieria en tecnologias de la informacion y las comunicaciones esta conformada por el conjunto de recursos necesarios para el procesamiento de la informacion asi como la administracion o gestion de las comunicaciones. Crea, diseña, implementa y administra sistemas de computo relacionados con las Tecnologias de Informacion y las comunicaciones, asi como de redes de computo, bases de datos para la toma de decisiones y dispositivos con software empotrado.

¿TICS ES PARA TI?

Preguntate si tienes: Interes por las redes de computadoras y en aprender a desarrollar aplicaciones para dispositivos moviles, gusto o habilidad para las ciencias exactas (Matematicas, calculo integral, etc.) capacidad intelectual y habilidades de razonamiento logico utilizando sistemas de computo.
